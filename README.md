An AI-powered fitness companion mobile application that generates personalized meal and workout plans based on user goals, dietary preferences, and fitness level. Built using Android Studio with Java and integrated with AI features.

A personal portfolio website featuring parallax scrolling effects, animations, and responsive design. Built to showcase my projects and skills with a modern user interface.
